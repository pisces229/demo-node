# pnpm (https://github.com/pnpm/pnpm)

```bash
# check node version
node -v
# install pnpm
npm install -g pnpm
# check pnpm version
pnpm -v
# install corepack（recommandNode.js 16.13+）
npm install -g corepack
corepack --version
corepack enable
# init pnpm
pnpm init
# install all packages
pnpm install
# add package
pnpm add [--save-dev] <package_name>
# update package
pnpm update [--save-dev] <package_name>
# remove package
pnpm remove [--save-dev] <package_name>
```
